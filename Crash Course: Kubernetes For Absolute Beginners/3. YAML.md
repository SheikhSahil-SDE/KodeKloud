#YAML
- Key-Value Pair
  <br>Sample:
```
          Fruit: Apple
          Vegetable: Carrot
          Liquid: Water
          Meat: Beef
```
  
- Array/Lists
  <br>Sample:
```
        Fruits:
          - Apple
          - Orange
          - Banana

        Vegetable:
          - Carrot
          - Potato
          - Tommato
```
  
- Dictionary/Map
  <br>Sample:
```
        Banana:
          - Calories: 105
          - Fat: 0.4g
          - Carbs: 27g

        Grapes:
          - Calories:
          - Fat:
          - Carbs:

```

#YAML-Advanced
- Key-Value/Dictionary/Lists
    <br>Sample:
```
        Fruits:
            Banana:
              - Calories: 105
              - Fat: 0.4g
              - Carbs: 27g
    
            Grapes:
              - Calories:
              - Fat:
              - Carbs:
```

- Dictionary in Dictionary
<br>Sample:
```
      Color: blue
      Model:
        Name: Corvette
        Year: 1995
      Transmission: Manual
      Price: $20,000
```
